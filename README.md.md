# مركز المعلومات والتوثيق السياحي الليبي

هذا الموقع الرسمي لمركز المعلومات والتوثيق السياحي الليبي، يهدف إلى تقديم أحدث الدراسات، الإحصاءات، والخدمات الذكية لتطوير القطاع السياحي في ليبيا.

## المميزات

- عرض إحصائيات حية للقطاع السياحي
- معرض صور سياحية يحتوي على 44 صورة متنوعة
- خدمات إلكترونية متكاملة
- مساعد افتراضي ذكي للرد على استفسارات الزوار
- تصميم متجاوب يعمل على جميع الأجهزة
